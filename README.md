# dd-lisp-wip
Public-safe WIP AutoCAD duct routing routine for CAD/BIM workflow automation.

## Sheet Setup Utilities
- **resizevp.lsp**: viewport sizing and repositioning without disturbing model scale.
- **centervp.lsp**: centers a viewport on a selected model-space rectangle.
- **mtexttransfer.lsp**: transfers MTEXT from one object to another and deletes the source.
